Medical Clinic System

The "Medical Clinic System" application has been created to streamline the process of managing patient data in a medical clinic. It is a simple yet functional application that allows adding, editing, deleting, sorting and listing patients.

Functionalities:

  • Add Patient: Users can add new patients to the database through a form containing fields for personal data (first name, last name, PESEL) and address (street, city, zip code).
  • Edit Patient: There is an option to edit the data of an existing patient, allowing for the update of information, such as changing the address.
  • Delete Patient: There is an option to delete unnecessary or incorrectly added patient entries from the database.
  • List Patients: The application enables displaying a list of all patients in the form of a table, presenting their basic personal and address data.
  • Sorting: There are buttons with which you can sort the current patient list in a way that suits you best, e.g. by index, by first name, etc.

Required Patient Data:

  • First Name
  • Last Name
  • PESEL (Personal Identification Number)
  • Street
  • City
  • Zip Code

Data Storage:

Patient data is stored in a database, ensuring permanent and secure storage of information. In this application I used the SQLite3 database.
